What is Backy?

This gem helps you to easly backup your PostgreSQL or Mysql database to .sql file. You can also use it with capistrano.

Backy is invoking direct database operations, thatswhy you need to be a database owner to use gem rake tasks.

Installation

To install gem add this line to your Gemfile

gem 'backy'

and run

bundle install

Usage

To backup your database simply run a rake task in your console.

rake backy:dump

This will create a backy.sql file in the /db directory

To rebuild your database type:

rake backy:load
